```

要確認階段式主題是否已經清楚
```
# 學習主題
## A.基礎程式設計
```
1 輸出與輸入
   格式化的輸出與輸入
2.基本資料型態  ==> 進階型資料結構
   數值型資料型態
   字串或字元型資料型態
3.運算子、運算式與敘述
4.選擇性敘述 if   switch
5.迴圈 for   while     do.. while
6.函數
    遞迴函數
7.陣列(不使用指標)
8.字串(字元陣列)與字串處理(不使用指標)
```
## 進階程式設計
```
9.指標(pointer)
10. 陣列(使用指標)
11. 字串處理(使用指標)
12. 結構與其它資料型態
13. 動態記憶體配置與鏈結串列
```
## 應用程式設計
```
14. 檔案處理
16. 位元處理
```
